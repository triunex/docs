## Overview

NSP uses cloud-verified API keys. All API keys are generated and managed through the Nelieo platform dashboard at [platform.nelieo.com](https://platform.nelieo.com).

When your agent makes requests to the local NSP daemon with `X-Substrate-Key: nsp_live_sk_...`, the daemon verifies the key with the Nelieo cloud platform (`platform.nelieo.com/api/verify`) and caches the validation in memory for sub-millisecond subsequent requests.

---

## Generating API Keys

1. Go to [platform.nelieo.com](https://platform.nelieo.com)
2. Sign in to your Nelieo account
3. Click **Create New API Key**
4. Assign a label (e.g. `prod-agent-gmail` or `dev-local`)
5. Copy your newly generated key

Your key will be formatted like:
```text
nsp_live_sk_a1b2c3d4e5f6789...
```

<Warning>
  Copy your key immediately upon generation — it is shown only once. If lost, you can revoke it and generate a new key from the platform console.
</Warning>

---

## Key Management & Revocation

You can manage all active keys directly from [platform.nelieo.com](https://platform.nelieo.com):

- **Labeling:** Organize keys by agent name, environment, or team member.
- **Revocation:** Disable or delete a key instantly if it is compromised. Once revoked in the cloud console, the local daemon will reject any requests using that key.
- **Audit Logs:** Monitor key creation dates and last-used timestamps across your agents.

---

## Key Rotation

To rotate an API key safely without downtime:

1. Generate a new key at [platform.nelieo.com](https://platform.nelieo.com) with a label like `prod-agent-v2`.
2. Update your agent environment variable (`NSP_API_KEY=nsp_live_sk_<new>`) or application configuration.
3. Test your agent to ensure calls succeed:
   ```bash
   curl -H "X-Substrate-Key: nsp_live_sk_<new>" http://localhost:7842/substrate/v1/apps
   ```
4. Revoke or delete the old key in the Nelieo platform console.

## Using Keys in Code

### Python SDK

```python
import nelieo_nsp as nsp

# Option 1: Pass the key directly
async with nsp.NSPClient(api_key="nsp_live_sk_a1b2c3d4...") as client:
    apps = await client.list_apps()

# Option 2: Set the NSP_API_KEY environment variable (Recommended)
# export NSP_API_KEY=nsp_live_sk_a1b2c3d4...
async with nsp.NSPClient() as client:
    apps = await client.list_apps()
```

### cURL / REST API

```bash
curl -H "X-Substrate-Key: nsp_live_sk_a1b2c3d4..." \
     http://localhost:7842/substrate/v1/apps
```
