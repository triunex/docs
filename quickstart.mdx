## Prerequisites

- **Windows 10/11 x64** (macOS/Linux support in beta)
- **Python 3.11+** for the SDK
- **Google Chrome** for V8/JavaScript app probing

---

## Step 1 — Install the Daemon

Download the latest `Nelieo NSP Setup.exe` from [platform.nelieo.com/download](https://platform.nelieo.com/download) and run it.

The installer:
- Registers `axon-daemon.exe` as a Windows Service (auto-starts on login)
- Places the daemon at `C:\Program Files\Nelieo\axon-daemon.exe`
- Creates a default config at `C:\ProgramData\Nelieo\axon.toml`
- Starts the daemon immediately on port **7842**

<Tip>
  You can also run the daemon manually for development:
  ```bash
  axon-daemon.exe run
  ```
</Tip>

### Verify the daemon is running

```bash
curl http://localhost:7842/health
```

```json
{
  "status": "ok",
  "version": "1.0.0",
  "uptime_secs": 42,
  "tracked_processes": 0,
  "auth_enabled": false
}
```

## Step 2 — Configure Chrome for V8 Probing

For NSP to extract semantic state from JavaScript apps (Gmail, Slack, VS Code, etc.), Chrome must be launched with the CDP remote debugging port enabled.

**Windows (Command Prompt):**
```cmd
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
```

**Windows (PowerShell):**
```powershell
Start-Process "chrome.exe" --ArgumentList "--remote-debugging-port=9222"
```

<Warning>
  You must **close all existing Chrome windows first**, then relaunch with this flag. Chrome only supports one debugging port per user profile.
</Warning>

Once Chrome is running, open Gmail (`mail.google.com`) in any tab.

## Step 6 — Write Your First Agent

```python
import asyncio
import nelieo_nsp as nsp

async def gmail_agent():
    async with nsp.NSPClient() as client:
        # Attach to Gmail by name (no PID needed)
        session = await client.attach("Gmail")
        
        # Read state with typed helpers
        unread = session.get_int("inbox.unread_count")
        print(f"You have {unread} unread emails")
        
        # Read the first email's subject
        subject = session.get_str("inbox.emails[0].subject")
        sender = session.get_str("inbox.emails[0].from")
        print(f"Latest: '{subject}' from {sender}")
        
        # Execute an action (with verification)
        thread_id = session.get_str("inbox.emails[0].thread_id")
        result = await session.execute(
            "send_reply",
            parameters={
                "thread_id": thread_id,
                "body": "Thanks, I'll review this shortly."
            },
            verify=True,
            verify_expression="inbox.emails[0].is_read == true",
        )
        
        print(f"Reply sent: {result.success} in {result.latency_ms}ms")

asyncio.run(gmail_agent())
```

**Output:**
```
You have 47 unread emails
Latest: 'Q3 Performance Report' from alice@example.com
Reply sent: True in 312ms
```
