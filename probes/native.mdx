## Release Notice

Native C/C++ application probing (for processes without managed runtimes such as C/C++, Rust, and Go binaries) is currently in final testing and will be released **next month**.

Current NSP releases support:
- **V8 / JavaScript** (Chrome, Electron, Node.js)
- **JVM / Java** (Java desktop, Spring, enterprise JVM services)
- **CLR / .NET** (WinForms, WPF, .NET 4.x/6+ apps)
