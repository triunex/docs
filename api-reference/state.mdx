---
title: "GET /state/{pid}"
description: "Get the full semantic state snapshot for a tracked process."
---

## Endpoint

```http
GET /substrate/v1/state/{pid}?fresh=false
Authorization: X-Substrate-Key: <key>
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `pid` | `integer` | OS process ID from `GET /apps` |

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `fresh` | `boolean` | `false` | If `true`, triggers a fresh probe cycle before returning. Adds 50ms–90s latency. |

---

### Top-Level Fields

| Field | Type | Description |
|-------|------|-------------|
| `capture_id` | `string` (UUIDv7) | Time-ordered unique capture ID |
| `app_id` | `string` | Stable app identifier |
| `app_name` | `string` | Human-readable name |
| `pid` | `integer` | OS process ID |
| `runtime` | `string` | Runtime kind |
| `schema_version` | `string` | Semantic schema version (semver) |
| `captured_at` | `datetime` | ISO 8601 timestamp |
| `confidence` | `float` | 0.0–1.0 probe quality |
| `context` | `AppContext` | Navigational context |
| `objects` | `object` | Dot-notation state key-value map |
| `actions` | `object` | Available action schemas |
| `probe_metadata` | `object` | Internal telemetry |

### `probe_metadata` Fields

| Field | Type | Description |
|-------|------|-------------|
| `capture_latency_ms` | `integer` | Time to read from runtime (ms) |
| `labeling_latency_ms` | `integer` | Time for semantic labeling (ms) |
| `total_latency_ms` | `integer` | End-to-end wall time (ms) |
| `raw_object_count` | `integer` | Objects before filtering |
| `filtered_object_count` | `integer` | Objects after filtering |
| `cache_hit` | `boolean` | Whether schema was read from cache |
| `probe_tier` | `string` | `aso_fast` / `heap_snapshot` / `jvmti` / `clr_mmf` |

## Examples

<CodeGroup>
```bash cURL
curl -H "X-Substrate-Key: nsp_live_sk_..." \
     http://localhost:7842/substrate/v1/state/12847

curl -H "X-Substrate-Key: nsp_live_sk_..." \
     "http://localhost:7842/substrate/v1/state/12847?fresh=true"
```

```python Python SDK
# Cached (fast)
state = await client.get_state(12847)
print(state.confidence)
print(state.objects.get("inbox.unread_count"))

# Fresh (slower)
state = await client.get_state(12847, fresh=True)

# Via session
session = await client.attach("Gmail")
state = await session.refresh()
```
</CodeGroup>

## Error Responses

| Status | Code | Description |
|--------|------|-------------|
| `404` | `not_found` | PID not tracked |
| `503` | `probe_not_ready` | Cold probe still in progress |
| `504` | `probe_timeout` | Fresh probe timed out |
